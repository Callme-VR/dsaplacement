# 🔗 Two Pointers

The Two Pointers technique is highly efficient for searching elements in sorted arrays or manipulating data in a linear fashion.

---

## 📂 Problems List

| Problem Name | Language | Description |
| :--- | :--- | :--- |
| **3Sum** | C++ | Find all unique triplets that sum to zero. |
| **4Sum** | C++ | Find all unique quadruplets that sum to a target. |
| **Merge Sorted Array** | C++ | Merge two sorted arrays in-place. |
| **Merge String Alt** | C++ | Merge two strings alternately. |
| **Remove Duplicates** | C++ | Remove duplicates from a sorted array. |
| **Reverse String** | C++ | Reverse a string in-place. |
| **Two Sum II** | C++ | Two Sum problem specifically for sorted arrays. |
| **Valid Palindrome** | C++ | Check if a string is a palindrome. |
| **Valid Palindrome II** | C++ | Check if a string can be a palindrome after deleting at most one character. |

---

### 💡 Key Concepts
- Using two indices (left/right) to narrow down the search space.
- Handling sorted data for O(n) solutions.
- In-place modifications to save memory.
