# 🪟 Sliding Window

The Sliding Window technique is used to solve problems that involve searching for a specific range of elements within a sequence.

---

## 📂 Problems List

| Problem Name | Language | Description |
| :--- | :--- | :--- |
| **Buy & Sell Stock** | C++ | Find maximum profit with one transaction. |
| **Contains Duplicate II** | C++ | Check for duplicates within a certain distance $k$. |
| **Find $k$ Closest Elements** | C++ | Find $k$ elements closest to a value $x$. |
| **Longest Repeating Char Replacement** | C++ | Max length of substring with replaced characters. |
| **Longest Substring Without Repeating** | C++ | Find length of longest substring with unique chars. |
| **Minimum Size Subarray Sum** | C++ | Smallest subarray with sum $\ge$ target. |
| **Minimum Window Sliding** | C++ | Smallest substring containing all characters of $T$. |
| **Permutation in String** | C++ | Check if $S_2$ contains a permutation of $S_1$. |
| **Sliding Window Maximum** | C++ | Find maximums in each sliding window of size $k$. |

---

### 💡 Key Concepts
- Constant or variable sized windows.
- Frequency counting within a window.
- Optimizing $O(n^2)$ problems to $O(n)$.
