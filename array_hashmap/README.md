# 📊 Array & Hashmap

This folder contains solutions for problems related to Array and Hashing techniques.

---

## 📂 Problems List

| Problem Name | Language | Description |
| :--- | :--- | :--- |
| **Duplicate Integers** | C++, Python | Check if an array has duplicates. |
| **Encode & Decode** | C++ | Encode and decode a list of strings. |
| **Group Anagrams** | C++ | Group strings that are anagrams of each other. |
| **LCS (Longest Consecutive Sequence)** | C++ | Find the longest consecutive sequence in an unsorted array. |
| **Product of Array Except Self** | C++ | Compute products of all elements except the current one. |
| **Two Sum** | C++ | Find two numbers that sum up to a target. |
| **Valid Anagram** | Python | Determine if two strings are anagrams. |
| **Valid Sudoku** | C++ | Validate a 9x9 Sudoku board. |

---

### 💡 Key Concepts
- Hash Maps/Sets for O(1) lookups.
- Sorting-based optimizations.
- Frequency counting.
