# 🔢 Bit Manipulation

Bit manipulation involves operating on binary representations of data to solve problems efficiently.

---

## 📂 Problems List

| Problem Name | Language | Description |
| :--- | :--- | :--- |
| **Add Two Integers** | C++ | Summing two numbers without using the $+$ operator. |
| **Count Bits** | C++ | Count the number of set bits for numbers $0$ to $n$. |
| **Number of 1 Bits** | C++ | Hamming weight calculation. |

---

### 💡 Key Concepts
- Bitwise AND (`&`), OR (`|`), XOR (`^`).
- Left and right shifting (`<<`, `>>`).
- $O(1)$ space and $O(\log n)$ time solutions.
