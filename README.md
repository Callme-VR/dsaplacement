## 🚀 Getting Started

### Prerequisites

Ensure you have the following installed:
- [GCC/G++](https://gcc.gnu.org/) for C++ files.
- [Python 3.x](https://www.python.org/) for Python scripts.

### Running a C++ Solution

```bash
cd folder_name
g++ problem_name.cpp -o solution
./solution
```

### Running a Python Solution

```bash
cd folder_name
python problem_name.py
```
